0.10.1b [2020-01-30]
+ Добавлена возможность отключения PVP для друзей.

0.10.0b [2020-01-17]
+ Доработан дизайн меню.
+ Максимальное количество символов для заметки о пользователе в списке сокращено до 60.
+ Добавлен кулдаун в 1 секунду для отправки пакетов - операций в меню списка друзей.
+ Обновлены параметры конфигурации:
 Клиент 
 - "enable_friendslist_key" [boolean] - разрешить добавление клавиши "открыть список друзей"
 
 Сервер 
 - "friend_request_expire_time_seconds" [int] - время активности запроса в друзья в секундах
 
 - "max_friends_amount" [int](синхронизируется) - максимальное количество друзей
 - "max_ignored_amount" [int](синхронизируется) - максимальное количество игнорируемых пользователей

+ Добавлены команды:
 Клиент [/oxygenc]
 - Открыть список друзей: /oxygenc friendslist
 - Сбросить данные клиента: /oxygenc friendslist -reset-data

+ Добавлены привилегии:
 - "friendslist:preventIgnore" [id - 600, type - BOOLEAN] - возможность запрета добавления игрока с этой привилегией в список игнорируемых.
 - "friendslist:maxFriendsAmount" [id - 601, type - INT] - определяет макс. число друзей.
 - "friendslist:maxIgnoredAmount" [id - 602, type - INT] - определяет макс. число игнорируемых пользователей.

+ Добавлены клиентские настройки:
 Общие
 - возможность открыть список друзей через общее меню (Oxygen Menu)
 
 Интерфейс
 - расположение меню
